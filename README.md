# VALTRIXAPP

Premium Project Management Platform

## About

VALTRIXAPP is a modern task management and business intelligence platform designed to help teams track productivity, manage tasks efficiently, and make data-driven decisions.

## Key Features

- **Business Dashboard**: Real-time metrics including MRR, Churn Rate, ROI, and Growth Tracking
- **Task Management**: Comprehensive task tracking with priorities, statuses, and filtering
- **Calendar View**: Visual scheduling with priority-based task indicators
- **Analytics**: Detailed insights into team productivity and business performance
- **Team Collaboration**: User assignments, comments, and file attachments

## Tech Stack

- React 19 + TypeScript
- Vite for build tooling
- Tailwind CSS + shadcn/ui components
- Zustand for state management
- React Router for navigation
- Recharts for data visualization

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
